# Non-LDS role search scope

The next bounded search covers only `attn_qo` (512x4096x4096), `ffn_down`
(512x4096x12288), and `attn_kv` (512x1024x4096). It reuses
`extra/qk/prefill_v2_schedule_search.py` and searches its existing
`UPCAST_M`, `UPCAST_N`, `LOCAL`, and `UNROLL` grid. No candidate is admitted and
the gate/up buffer2 default is untouched.

Commands are generated by `extra/qk/prefill_nonlds_role_search_scope.py`, for
example:

```bash
PYTHONPATH=. DEV=AMD TC=1 ALLOW_DEVICE_USAGE=1 \
python3 extra/qk/prefill_v2_schedule_search.py \
  --shapes 4096,4096 --out bench/prefill-nonlds-search/attn_qo.json
```

Each result must separately pass correctness, no-spill/resource checks, and
the existing full-kernel candidate admission protocol before any runtime
promotion. This scope is host-testable and does not require a new emitter.
